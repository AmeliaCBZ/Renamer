# RenameMe
Aplicación de escritorio creada específicamente para renombrar archivos de facturas de electricidad de CEPM según el número de contrato indicado en la factura. El sistema lee estos dígitos de cada archivo y los renombra según ellos.

Desktop application created specifically to rename CEPM electricity bill files according to the contract number indicated on the bill. The system reads these digits from each file and renames them accordingly.
